# Payment System using Polymorphism (Python)

This is a simple Python project that demonstrates the concept of **Polymorphism (Method Overriding)** using a Zomato-like payment system.

## Features
- Base class: PaymentSystem
- Child classes: CreditCard, UPIPayment
- Demonstrates runtime polymorphism
- Displays customer payment summary

## Concepts Used
- OOP (Object-Oriented Programming)
- Inheritance
- Polymorphism (Method Overriding)

## Example Output
- Payment done using Credit Card
- Payment done through UPI
- Customer name displayed in summary
